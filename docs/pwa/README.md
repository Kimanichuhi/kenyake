# PWA and Offline

## Install behavior
SafariSync supports PWA installation.
- The install banner appears when the browser fires beforeinstallprompt.
- The /install page provides manual steps for Android, iOS, and desktop.

## When install is not available
The browser may hide the install prompt if:
- The site is not served over HTTPS.
- The app is already installed.
- The browser does not support install prompts.

## Service worker
- The service worker is generated by Vite PWA.
- Static assets are cached for offline use.
- Runtime caching includes map tiles and Google Fonts.

## Offline maps
- Tiles are downloaded and cached per region.
- Download metadata is stored in IndexedDB.

## Destination packs
- Destination packs store experiences, accommodations, guides, and food listings.
- Packs are stored in IndexedDB for offline use.

## Low Data Mode
- Low Data Mode is saved in IndexedDB.
- The status bar exposes quick visibility and sync state.
