# SafariSync User Guide

## What SafariSync is
SafariSync is a Kenya tourism intelligence platform that combines destinations, community content, AI trip planning, and offline tools.

## Getting started
- Open the site in your browser.
- Create a profile on the Onboarding page if you want personalization.
- Sign in to unlock features like wildlife reporting and profile updates.

## Main areas
- Destinations: browse counties and destination details.
- Experiences: curated activities and tours.
- Wildlife: live sightings plus AI wildlife intelligence.
- Community: local communities and profiles.
- Guides: connect with local guides.
- Events: cultural events and festivals.
- Food: food listings and dining options.
- Safety: travel safety guidance.
- Impact: sustainability and community impact.
- Nomads: resources for remote workers.
- Marketplace: local products and services.
- Transport: routes and travel details.
- Heritage: cultural heritage highlights.

## AI Trip Planner
The Trip Planner provides conversational itinerary planning.
- Set your trip profile: group size, budget, fitness, dietary needs, interests, and dates.
- Use quick actions for common planning tasks.
- Ask for day-of suggestions, budget tips, and wildlife timing.

## Wildlife Intelligence
Wildlife Intelligence combines live sightings with AI insights.
- Live Feed shows recent sightings on a map.
- Species, Migration, Big Five, Birding, and Parks tabs provide AI-generated intelligence.
- Reporting a sighting requires sign-in.

## Offline and low data tools
SafariSync includes offline-first features.
- Offline Settings lets you download maps and destination packs.
- Low Data Mode reduces media usage and auto-loading.
- Pending actions are queued and synced when you are back online.

## Install the PWA
You can install SafariSync as an app.
- Visit the Install page at /install.
- Follow device-specific steps for Android, iOS, or desktop.
- If the install prompt is not available, use the manual steps shown on the page.

## Troubleshooting
- Install prompt not appearing: use HTTPS, close and reopen the tab, or use the manual steps.
- Offline downloads not visible: open Offline Settings and refresh the list.
- Sync stuck: open Offline Settings and tap Sync Now.

## Support
If a feature does not behave as expected, share a screenshot and the page URL with the team.
